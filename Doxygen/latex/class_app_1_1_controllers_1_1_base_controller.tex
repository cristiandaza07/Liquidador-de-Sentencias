\doxysubsection{Descripción detallada}
Class Base\+Controller

Base\+Controller provides a convenient place for loading components and performing functions that are needed by all your controllers. Extend this class in any new controllers\+: class Home extends Base\+Controller

For security be sure to declare any new methods as protected or private.
