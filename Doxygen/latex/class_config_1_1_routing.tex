string \$override404 = null}

Sets the class/method that should be called if routing doesn\textquotesingle{}t find a match. It can be the controller/method name like\+: Users\textbackslash{}index

This setting is passed to the Router class and handled there.

If you want to use a closure, you will have to set it in the routes file by calling\+:

\$routes-\/\texorpdfstring{$>$}{>}set404\+Override(function() \{ // Do something here \});

Example\+: public \$override404 = \textquotesingle{}App\textbackslash{}\+Errorsshow404\textquotesingle{};
