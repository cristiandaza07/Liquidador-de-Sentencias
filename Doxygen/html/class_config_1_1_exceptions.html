◆ $deprecationLogLevel

string $deprecationLogLevel = LogLevel::WARNING

LOG LEVEL THRESHOLD FOR DEPRECATIONS

If $logDeprecations is set to true, this sets the log level to which the deprecation will be logged. This should be one of the log levels recognized by PSR-3.

The related Config\Logger::$threshold should be adjusted, if needed, to capture logging the deprecations.

◆ $logDeprecations

bool $logDeprecations = true

LOG DEPRECATIONS INSTEAD OF THROWING?

By default, CodeIgniter converts deprecations into exceptions. Also, starting in PHP 8.1 will cause a lot of deprecated usage warnings. Use this option to temporarily cease the warnings and instead log those. This option also works for user deprecations.
